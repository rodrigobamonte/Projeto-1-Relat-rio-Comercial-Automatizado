# 📊 Projeto 01 - Relatório Comercial Automatizado
## Sobre o Projeto
Este projeto foi desenvolvido durante meus estudos de **Python** e **Análise de Dados**.
O objetivo foi automatizar o cálculo de indicadores comerciais (KPIs), simulando um cenário empresarial utilizando dados fictícios.
O projeto foi construído com foco na prática de lógica de programação e na aplicação de conceitos utilizados na rotina de um analista de dados.

## Objetivos do Projeto

- Praticar lógica de programação utilizando Python.
- Automatizar o cálculo de indicadores comerciais.
- Aplicar conceitos fundamentais de análise de dados.
- Desenvolver um projeto para compor meu portfólio.

- ## Indicadores Calculados

Neste projeto foram desenvolvidos os cálculos dos seguintes indicadores comerciais:

- Taxa de Ativação das Agências
- Percentual de Agências com Venda
- Taxa de Conversão
- Ticket Médio
- Leads Não Convertidos
- Receita Média por Agência com Venda

- ## Conceitos de Python Aplicados

Durante o desenvolvimento deste projeto foram utilizados os seguintes conceitos:

- Variáveis
- Operadores matemáticos
- Operadores relacionais
- Estruturas condicionais (`if`, `elif` e `else`)
- Cálculos com porcentagens
- Formatação de saída utilizando `f-string`

- ## 🛠️ Tecnologias Utilizadas

- Python
- Google Colab
- GitHub

- ## 📌 Observação

Todos os dados utilizados neste projeto são fictícios e foram criados exclusivamente para fins de estudo, prática de programação e desenvolvimento de portfólio.
